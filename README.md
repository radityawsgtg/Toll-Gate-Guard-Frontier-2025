This project is a simulation of an Automated Toll Gateway Payment system, developed as part of the Computational Thinking assignment for STEI-K ITB 2025. 
It aims to mimic real-world payment processes while integrating gamification elements to enhance user engagement. All visual and functional assets used
in this project are original and self-created

# Toll Gateway Automatic Payment Simulation

Project ini dibuat untuk memenuhi tugas **Komputasi STEI-K ITB 2025**.

## Deskripsi
Simulasi ini dirancang untuk meniru proses pembayaran gerbang tol otomatis dengan sentuhan gamifikasi agar pengalaman pengguna lebih interaktif.

## Fitur Utama
* **Realistic Payment Mimicry**: Simulasi alur transaksi tol yang akurat.
* **Gamification**: Elemen permainan untuk meningkatkan engagement.
* **Original Assets**: Semua aset visual dan fungsional dibuat secara mandiri.
